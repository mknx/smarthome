    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-raumfeld-plugin" class="anchor" href="#raumfeld-plugin" aria-hidden="true"><span class="octicon octicon-link"></span></a>Raumfeld Plugin</h1>

<p>Thanks to tfeldmann for his raumfeld code: <a href="https://github.com/tfeldmann/python-raumfeld">https://github.com/tfeldmann/python-raumfeld</a></p>

<p>This is a prototyp for a simple raumfeld integration. The current version
supports start and stop playing stream in a specific zone. 
I wrote this plugin to start and stop radio stream in my bathroom and kitchen using a knx sensor</p>

<h1><a id="user-content-requirements" class="anchor" href="#requirements" aria-hidden="true"><span class="octicon octicon-link"></span></a>Requirements</h1>

<p>Plugin for Teufel Raumfeld compontents. See <a href="https://www.raumfeld.com/">https://www.raumfeld.com/</a></p>

<h2><a id="user-content-supported-hardware" class="anchor" href="#supported-hardware" aria-hidden="true"><span class="octicon octicon-link"></span></a>Supported Hardware</h2>

<ul>
<li>all Raumfeld Speaker</li>
</ul>

<h1><a id="user-content-configuration" class="anchor" href="#configuration" aria-hidden="true"><span class="octicon octicon-link"></span></a>Configuration</h1>

<h2><a id="user-content-pluginconf" class="anchor" href="#pluginconf" aria-hidden="true"><span class="octicon octicon-link"></span></a>plugin.conf</h2>

<pre>[raumfeld]
    class_name = Raumfeld
    class_path = plugins.raumfeld
</pre>

<h2><a id="user-content-itemsconf" class="anchor" href="#itemsconf" aria-hidden="true"><span class="octicon octicon-link"></span></a>items.conf</h2>

<pre>[raumfeld]
    [[bad]]
        type = bool
        name = Badezimmer Musik
        device_name = Bad
        stream_url = http://rb-mp3-m-bremenvier.akacast.akamaistream.net/7/23/234437/v1/gnl.akacast.akamaistream.net/rb-mp3-m-bremenvier
        enforce_updates = true
        knx_dpt = 1
        knx_listen = 1/0/18
        knx_send = 1/0/18
        knx_init = 1/0/20
        knx_reply = 1/0/20
    
    [[kueche]]
        type = bool
        name = Küche Musik
        device_name = Kueche
        stream_url = http://rb-mp3-m-bremenvier.akacast.akamaistream.net/7/23/234437/v1/gnl.akacast.akamaistream.net/rb-mp3-m-bremenvier
        enforce_updates = true
        knx_dpt = 1
        knx_listen = 0/0/36
        knx_send = 0/0/36
        knx_init = 0/0/37
        knx_reply = 0/0/37
</pre>

<h3><a id="user-content-device_name" class="anchor" href="#device_name" aria-hidden="true"><span class="octicon octicon-link"></span></a>device_name</h3>

<p>Raumfeld Zone/Hörzonen name</p>
